##########################################
Contraction Interface and L2 Optimization
##########################################

In diesem Assignment wird ein hochstufiges Konfigurations-Interface für
Tensor-Kontraktionen entworfen, ein Optimizer implementiert, der diese
Konfigurationen transformiert, und das Ganze genutzt, um einen
L2-optimierten cuTile-Kernel abzuleiten und zu benchmarken.
